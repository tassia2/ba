\subsection{Pointers}
\begin{enumerate}
\item use Smart pointers like \code{std::unique\_ptr}, \code{std::shared\_ptr}, \code{std::weak\_ptr} when dynamically allocint memory through \code{new}
\item use \code{nullptr} instead of \code{NULL} or \code{0} for setting a pointer to zero
\end{enumerate}
